# installation

```bash

# langchain: for core langchain implementation
# langchain-community: for community langchain implementations
# langchain-ollama: for Ollama langchain implementation
# langchain-chroma: for Chroma vectore store
# pypdf2: for pdf file processing
# unstructured: for unstructured file processing
# langchain-openai: for openai langchain implementation
# bs4: for html parsing

> pip3 install langchain langchain-community langchain-openai langchain-chroma pypdf2 unstructured bs4

> pip3 install -r requirements.txt

```
